개인정보 처리방침

최종 업데이트: 2026년 5월 14일

Gainsy(이하 "회사")는 이용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」을 준수합니다.

1. 수집하는 개인정보 항목

항목수집 목적보유 기간
이메일 주소회원 식별, 로그인회원 탈퇴 시 즉시 삭제
신체 측정 데이터 (키, 체중 등)건강 목표 추적회원 탈퇴 시 즉시 삭제
식단 기록영양 분석 및 목표 달성 지원회원 탈퇴 시 즉시 삭제
운동 기록운동 목표 추적회원 탈퇴 시 즉시 삭제
진행 사진신체 변화 기록회원 탈퇴 시 즉시 삭제
FCM 토큰푸시 알림 발송회원 탈퇴 시 즉시 삭제

2. 개인정보 수집 방법

  • 앱 회원가입 및 서비스 이용 과정에서 이용자가 직접 입력
  • iOS 기기에서 자동 수집 (FCM 토큰)

3. 개인정보의 제3자 제공

회사는 이용자의 개인정보를 원칙적으로 외부에 제공하지 않습니다. 단, 서비스 제공을 위해 아래 수탁업체에 처리를 위탁합니다.

수탁업체위탁 업무
Amazon Web Services (AWS)서버 운영, 데이터 저장 (서울 리전)
Google Firebase푸시 알림 발송 (FCM), 앱 분석
OpenAI식단·운동 AI 추정 기능

4. 개인정보의 파기

회원 탈퇴 요청 즉시 모든 개인정보를 파기합니다. 앱 내 마이페이지 → 계정 삭제 기능을 통해 직접 탈퇴할 수 있습니다.

5. 이용자의 권리

  • 개인정보 열람, 수정, 삭제 요청 가능
  • 앱 내 계정 삭제 기능으로 즉시 탈퇴 가능
  • 만 14세 미만은 서비스를 이용할 수 없습니다

6. HealthKit 데이터

Gainsy는 Apple HealthKit 데이터를 광고, 마케팅 또는 제3자 판매 목적으로 사용하지 않습니다. HealthKit 데이터는 오직 앱 내 건강 목표 추적 기능에만 사용됩니다.

7. AI 기반 추정 기능

식단 및 운동 데이터의 AI 추정 결과는 의료적 진단이 아닙니다. 추정 결과에는 AI 생성 표시가 포함되며, 전문 의료인의 조언을 대체하지 않습니다.
