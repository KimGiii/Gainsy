## 배운 것 (Learned)

- **EXIF 제거 방법**: Java에서 `ImageIO.read()` → `ImageIO.write()`로 재인코딩하면 EXIF가 제거된다. `TwelveMonkeys ImageIO` 플러그인 없이 표준 JDK만으로 가능하지만, HEIC 포맷은 별도 처리가 필요하다.
- **GitHub Actions 서비스 컨테이너**: `services:` 블록에 PostgreSQL/Redis를 선언하면 job 시작 시 컨테이너가 자동으로 뜨고 `localhost`로 접근 가능하다. `options`의 `--health-cmd`로 헬스체크를 걸어야 race condition 없이 연결된다.
- **Terraform `depends_on` vs 암시적 의존성**: 리소스 속성에서 다른 리소스를 참조하면 Terraform이 자동으로 의존 관계를 파악한다. `depends_on`은 참조 없이 순서만 보장해야 할 때 쓴다.
