## 배운 것 (Learned)

- **`@JsonProperty("isBaseline")`**: Kotlin/Java에서 `boolean` 필드에 `is` prefix가 붙으면 Jackson이 직렬화 시 `is`를 제거한다(`isBaseline` → `baseline`). `@JsonProperty`로 명시하거나 `Boolean` 래퍼 타입을 쓰는 것이 안전하다.
- **GoalController `@RequestHeader(required = false)`**: Spring Security 없이 MockMvc `standaloneSetup`으로 컨트롤러 단위 테스트를 할 때 Authorization 헤더를 `required = false`로 선언하고 서비스 레이어에서 null 체크하는 패턴이 테스트와 실제 동작 모두에서 깔끔하다.
- **S3 presigned URL TTL**: 너무 짧으면 iOS에서 업로드 중 만료된다. 업로드용은 15분, 다운로드(signed)용은 15분으로 통일하되, 대용량 파일 업로드 시 TTL을 늘려야 한다.
